# Projeto_A3 — Sistema de Gestão de Investimentos

Sistema desktop em Java (Swing) com banco MySQL.

## Pré-requisitos
- JDK 21
- MySQL instalado e rodando
- NetBeans (com Maven)

## Como rodar
1. Clone o repositório.
2. No MySQL, execute o script `database/schema.sql` (cria o banco `investsys`
   e as tabelas, já com dados de teste).
3. Copie o arquivo `db.properties.example` e renomeie a cópia para `db.properties`.
4. Abra o `db.properties` e coloque o usuário e a senha do SEU MySQL.
5. Abra o projeto no NetBeans e rode (botão Run / F6). O Maven baixa as
   dependências sozinho.
